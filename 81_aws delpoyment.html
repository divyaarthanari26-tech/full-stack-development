AWS Deployment Process

Step 1: Create AWS Account

Sign up and log in to AWS Management Console.

Step 2: Launch EC2 Instance

Create an EC2 instance using Amazon Linux or Ubuntu.

Step 3: Connect to Server

Use SSH to connect:

ssh -i mykey.pem ec2-user@your-public-ip

Step 4: Install Web Server

sudo yum update -y
sudo yum install httpd -y
sudo systemctl start httpd

Step 5: Upload Website Files

Copy HTML files to:

/var/www/html/

Step 6: Access Website

Open your EC2 public IP address in a browser.
